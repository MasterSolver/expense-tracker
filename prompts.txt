Project Title  QuickSpend – Simple Expense & Budget Tracker  Problem  Most students and young professionals don’t track daily expenses because existing apps are:  Overcomplicated  Require signup/login  Take too much time to use  As a result, people lose control of their monthly spending.  Solution  QuickSpend is a lightweight web application that lets users:  Add expenses instantly  View total spending in real time  Categorize expenses  Delete entries  No login. No setup. No friction.  Key Features  Add expense (title, amount, category)  Live total calculation  Expense list with delete option  Fully functional in a browser  Tech Stack  Frontend: HTML, CSS, JavaScript  Backend: Node.js, Express  Storage: Temporary in-memory storage  Why Temporary Storage?  Faster performance  Zero setup  Ideal for quick demos and hackathons  Data resets on server restart (future scope: database integration)  Future Scope  Persistent database  Monthly budget limits  Category-wise charts  User accounts (optional)
